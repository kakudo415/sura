# Sura

Text editor. Sura Sura.

## Usage

### Running

```
cargo run "file path" "language server path"
```

### Keyboard commands

| Keys       | Function                 |
| ---------- | ------------------------ |
| Arrow keys | Move cursor              |
| Ctrl + N   | Go forward to next page  |
| Ctrl + B   | Go back to previous page |
| Ctrl + S   | Save                     |
| Ctrl + Q   | Quit                     |
